## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Google login

The login page supports Google Identity Services in the frontend.

1. Copy `.env.example` to `.env.local`.
2. Set `NEXT_PUBLIC_GOOGLE_CLIENT_ID` with your OAuth client id from Google Cloud Console.
3. Make sure the OAuth client allows the local dev origin, for example `http://localhost:3000`.
4. Commit `.env.example` to the repo, but keep `.env.local` untracked.

If the variable is missing, the Google button falls back to a disabled state so the app does not auto-login with a mock account.
